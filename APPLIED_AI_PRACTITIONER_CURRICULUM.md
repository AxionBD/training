## 📋 COURSE OBJECTIVES

By the end of this 5-week program, students will:

✅ Write production-quality Python code (OOP, modules, testing)  
✅ Perform data analysis using NumPy and Pandas  
✅ Build and evaluate machine learning models  
✅ Understand and implement deep learning concepts  
✅ Master prompt engineering and LLM applications  
✅ Build RAG systems and deploy AI applications  
✅ Understand AI ethics, bias, and responsible AI practices  
✅ Complete a portfolio-ready capstone project  
✅ Prepare for AI/Data Science job roles

## 🏆 ENTRY REQUIREMENTS & PREREQUISITES
